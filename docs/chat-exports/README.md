# chat-exports

**보조용** — SpecStory(`.specstory/history/`)가 기본 PC 간 채팅 동기화 방식입니다.

수동 Export가 필요할 때만:

1. Cursor → **Export Transcript**
2. `docs/chat-exports/YYYY-MM-DD-주제.md` 저장
3. `git commit` → `push`

PC 간 채팅 이력: [SESSION_HANDOFF.md](../onboarding/SESSION_HANDOFF.md) SpecStory 섹션 참고.

전체 환경 설정: [SETUP.md](../onboarding/SETUP.md)
